# Docs Archive

This folder stores archived documents kept for historical context and rollback safety.

## Archive Policy

1. Archive-first: uncertain or duplicate docs are moved here before deletion.
2. Keep discoverability: active docs should link here when content is intentionally archived.
3. No secrets: archived docs must not contain live tokens, keys, or production credentials.
